# Summary (摘要) Action 插件

## 概述

**Summary Action 插件** 是一款旨在直接在 OpenWebUI 中生成简洁准确的文本内容摘要的工具。

> **注意**: 本文档为占位符。暂时无法在仓库中找到该插件的源代码或详细文档。

## 功能

*   **文本摘要**: 自动生成对话或提供文本的摘要。
*   **可定制**: (可能具有可配置参数，详情待定)。

## 安装

1.  下载插件文件（如果可用）。
2.  进入 **管理面板** -> **设置** -> **插件**。
3.  上传文件并启用它。

## 使用方法

1.  在聊天中，选择 "Summary" 动作。
2.  插件将处理上下文并提供摘要。
