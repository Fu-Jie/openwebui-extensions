# Frequently Asked Questions (FAQ)

## General

### Do I need to know Python to use these plugins?
No, for most plugins, you just need to install them. However, if you want to customize them or develop your own, Python knowledge is required.

### Are these plugins safe?
All plugins in this repository are open source. You can inspect the code before installing. However, always exercise caution when installing third-party code.

## Plugins

### My plugin isn't showing up after upload.
*   Try refreshing the page.
*   Go to **Admin Panel** -> **Settings** -> **Plugins** and ensure the plugin is enabled.
*   Check if the filename ends with `.py`.

### The "Smart Mind Map" plugin isn't generating a map.
*   Ensure you have entered enough text for the model to analyze.
*   Check if the LLM you are using is responding correctly.

## Prompts

### How do I modify a prompt?
You can edit the text directly in the OpenWebUI prompt editor after importing it.
