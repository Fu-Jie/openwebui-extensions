# 快速入门

欢迎来到 OpenWebUI Extras 文档！本集合旨在通过强大的插件、优化的提示词和详细的开发指南来增强您的 OpenWebUI 体验。

## 本项目是什么？

本仓库包含：

*   **插件 (Plugins)**：扩展 OpenWebUI 功能的组件（Actions, Filters, Pipes）。
*   **提示词 (Prompts)**：精心策划的系统提示词库，涵盖编程、营销等多种任务。
*   **开发指南 (Development Guides)**：为开发者提供的构建自定义插件的综合文档。

## 如何使用本文档

*   **用户指南**：如果您只是想安装和使用插件，请从 [安装指南](installation.md) 开始。
*   **资源目录**：浏览 [插件](../plugins/index.md) 和 [提示词](../prompts/index.md) 部分，寻找适合您的工具。
*   **开发者指南**：如果您是开发者，请查看 [插件开发](../development/plugin-guide.md) 部分。
