# Getting Started

Welcome to the OpenWebUI Extras documentation! This collection aims to enhance your OpenWebUI experience with powerful plugins, refined prompts, and development guides.

## What is this project?

This repository contains:

*   **Plugins**: Extensions that add new capabilities (Actions, Filters, Pipes) to OpenWebUI.
*   **Prompts**: A curated library of system prompts for various tasks (Coding, Marketing, etc.).
*   **Development Guides**: Comprehensive documentation for developers to build their own plugins.

## How to use this documentation

*   **User Guide**: If you just want to install and use the plugins, start with the [Installation](installation.md) guide.
*   **Resource Catalog**: Browse the [Plugins](../plugins/index.md) and [Prompts](../prompts/index.md) sections to find tools that fit your needs.
*   **Developer Guide**: If you are a developer, check out the [Plugin Development](../development/plugin-guide.md) section.
