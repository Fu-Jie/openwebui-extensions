# 常见问题 (FAQ)

## 通用问题

### 我需要懂 Python 才能使用这些插件吗？
不需要，对于大多数插件，您只需要安装即可使用。但是，如果您想自定义它们或开发自己的插件，则需要 Python 知识。

### 这些插件安全吗？
本仓库中的所有插件都是开源的。您可以在安装前检查代码。但是，安装第三方代码时请始终保持谨慎。

## 插件相关

### 上传后我的插件没有显示。
*   尝试刷新页面。
*   转到 **管理面板** -> **设置** -> **插件**，确保插件已启用。
*   检查文件名是否以 `.py` 结尾。

### "智能思维导图" 插件没有生成导图。
*   确保您输入了足够的文本供模型分析。
*   检查您使用的 LLM 是否正常响应。

## 提示词相关

### 如何修改提示词？
导入后，您可以在 OpenWebUI 提示词编辑器中直接编辑文本。
